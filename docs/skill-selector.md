# 我该用哪个 skill？

| 我的需求 | 触发 |
|:---------|:-----|
| 我想写一篇短篇 | `/novel-assistant 写短篇` |
| 我想写一本长篇 | `/novel-assistant 写长篇` |
| 我想拆解一本小说学习 | `/novel-assistant 拆长篇` 或 `/novel-assistant 拆短篇` |
| 我想了解当前市场热点 | `/novel-assistant 长篇扫榜` 或 `/novel-assistant 短篇扫榜` |
| 我写完了想去 AI 味 | `/novel-assistant 去AI味` |
| 我写完想审稿 | `/novel-assistant 审查` |
| 我想给小说做封面 | `/novel-assistant 封面` |
| 我有一本已写好的小说想导入 | `/novel-assistant 导入小说` |
| 我想问"沈栀现在什么状态" | （自动）spawn `story-explorer` |
| 模糊意图（"帮我写小说"） | `/novel-assistant` 自动分发 |
