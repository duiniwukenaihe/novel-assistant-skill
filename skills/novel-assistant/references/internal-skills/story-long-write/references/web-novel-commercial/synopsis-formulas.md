# 简介公式

## 番茄
200 字内，含「主角身份 + 核心矛盾 + 金手指 + 卖点」
句式：「我 XXX（身份），XXX（背景设定），却 XXX（反转/困境）——直到 XXX（金手指/转折）」

## 起点
300 字内，侧重世界观 + 主角动机 + 升级路径
句式：「在 XXX 世界，XXX（背景），主角 XXX（身份）凭借 XXX（优势）踏上 XXX（目标）」

## 晋江
150 字内，情感线优先
句式：「XXX（人设）+XXX（关系设定）+/ 虐 / 甜 / 追妻」

## 七猫
250 字内，爽点密集
句式：「被 XXX 看不起 → XXX 反转打脸 → 走上 XXX 巅峰」

## 知乎
50 字内，反转前置
句式：「我以为 XXX，结果 XXX」

## 黑岩
200 字内，刺激优先
句式：「（危机/困境）→（觉醒/金手指）→（碾压/复仇）」

## 点众
200 字内，男频爽文
句式：「退婚/被弃/濒死 → 觉醒金手指 → 一路碾压」
