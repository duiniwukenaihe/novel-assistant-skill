# human-resonance-gate.md：人性共鸣与文字质感门

本门禁用于长篇章节契约、细纲、正文、回炉，也可供短篇流程和 agent 参考。它解决的不是“像不像 AI”，而是“人物是否像真实的人，剧情是否有情感重量，文字是否承载了场景里的选择和代价”。

## 核心判断

每一章或每一节都至少要推动三件事中的两件：

1. **剧情变化**：事件、证据、关系、信息差、风险或承诺发生改变。
2. **人物变化**：人物欲望、恐惧、认知、关系位置、选择边界或代价发生改变。
3. **情绪变化**：读者对人物、关系、冲突或未来期待的判断发生改变。

只完成情节点但没有人的情绪后果，判定为 `human_resonance_failed`。不得用去 AI 味、扩字数或标点修复替代。长篇回到 Chapter Contract 或细纲补“关系压力 + 角色选择 + 代价”；短篇回到 `设定.md` / `小节大纲.md` 补“软肋 + 生活物件 + 情绪债 + 主动选择”。

## 短篇写前增补项

短篇在进入正文前必须写清：

```text
主角最怕失去什么：
主角最不愿承认的软肋：
反派/对手抓住了什么真实关系压力：
读者会代入哪一种现实经验：
本篇最扎人的生活物件/数字/旧话：
主角第一次主动反击的代价：
选择代价：
中段让读者改判的人性细节：
结尾让读者记住的动作或物件：
```

任一项答不出，不得进入正文。

## 长篇 Chapter Contract 增补项

每章契约在情节点之外补充：

```text
本章读者应感到：
本章人物最在乎的东西：
本章关系压力：
本章不可撤回选择：
本章生活/场景质感锚点：
本章情绪兑现方式：
```

过渡章也必须有“小变化”：关系更近/更远、风险更清晰、认知更具体、承诺更重、误解更深、旧钩子被轻触或新代价出现。没有小变化的过渡章，视为结构未完成。

## 人性场景五件套

关键场景至少满足 3 项：

1. **关系压力**：亲情、婚姻、师徒、同门、主仆、职场、金钱、名声、生命安全、身份归属中至少一项正在压迫角色。
2. **具体物件/场景锚点**：账单、钥匙、饭盒、病历、令牌、旧衣服、门牌、符箓、武器、伤口、饭菜、泥地、雨声、灯火等；锚点必须推动情绪或证据，不做摆设。
3. **角色选择**：主角或关键配角做出一个无法完全撤回的动作。
4. **身体反应**：不是“他震惊/她心痛”，而是手、胃、喉咙、脚步、呼吸、视线、沉默、伤口、动作停顿等可见反应。
5. **关系后果**：本章结束后，谁和谁的关系、权力、信任、误解或债务发生变化。

## 文字质感规则

- 情绪可以直写，但后面必须接本场景独有动作、物件、身体反应或选择代价。
- 对话必须带利益和关系位置。每句台词都要能回答“他/她为什么此刻这么说”。
- 不写通用漂亮话。不要用“终于明白”“这意味着”“命运齿轮”“一切尽在不言中”替代场景。
- 不把读者该感受到的东西提前解释出来。让物件、动作、对话和选择自己完成情绪。
- 现实题材要有生活纹理；玄幻/修仙/武侠也要有场景纹理，不能只有境界、任务和设定名词。
- 爽点必须有情绪债。没有前面的羞辱、误解、损失、犹豫或代价，后面的打脸只会像流程。

## 失败信号

命中以下任一项，停止继续写下一章/下一节：

- 主角一直在等别人给证据、给任务或给台阶，没有主动动作。
- 反派只是“坏/强/蠢”，没有具体欲望、利益和手段。
- 章节只是在解释前因或搬运细纲，读者判断没有变化。
- 权威、系统、豪门、宗门、师父或外挂替主角完成主题。
- 情绪只靠标签和金句，没有生活物件、场景锚点和身体反应。
- 每章都有事件，但事件之间没有关系后果。
- 读完只知道“发生了什么”，不知道“这对这个人意味着什么”。

## 通过标准

```text
Human resonance gate:
- Verdict: pass / revise / hard fail
- 共鸣经验：
- 关系压力：
- 生活物件/场景锚点：
- 主角主动选择：
- 情绪债与兑现：
- 文字质感风险：
- 必改动作：
```

`revise` 时先改细纲 / Chapter Contract / `设定.md` / `小节大纲.md`，再局部重写正文。`hard fail` 时保留用户确认的好开头、好句子、好物件和已成立的事实，重建故事发动机。
