# human-resonance-gate.md：短篇人性共鸣与文字质感门

本门禁用于短篇构思、小节大纲、正文写作和回炉。它解决的不是“像不像 AI”，而是“有没有人的情感、关系压力、生活质感和可读的文字”。

## 核心判断

短篇不是把一个社会议题写完，也不是把证据链讲清楚。读者愿意读下去，是因为她能立刻感到：

- 这个人为什么难受、羞耻、心寒、害怕或想反击。
- 这个关系为什么不能一句话断掉。
- 这个选择为什么有代价。
- 这件事为什么像身边会发生的事。
- 这句话、这个物件、这个动作为什么扎人。

如果正文只是“不 AI、逻辑通、字数够”，但读者无法感到上述任一项，判定为 `human_resonance_failed`，回到 `设定.md` 和 `小节大纲.md` 修故事，不做表层润色。

## 写前必须回答

进入正文前，在 `设定.md` 或 `小节大纲.md` 中写清：

```text
主角最怕失去什么：
主角最不愿承认的软肋：
反派/对手抓住了什么真实关系压力：
读者会代入哪一种现实经验：
本篇最扎人的生活物件/数字/旧话：
主角第一次主动反击的代价：
选择代价：
中段让读者改判的人性细节：
结尾让读者记住的动作或物件：
```

任一项答不出，不得进入正文。

## 人性场景五件套

每个关键小节至少有 3 项：

1. **关系压力**：亲情、婚姻、职场、金钱、名声、身体安全、身份归属中至少一项正在压迫角色。
2. **具体物件**：账单、钥匙、电话、饭盒、病历、结婚证、群聊截图、旧衣服、门牌、红包等；物件必须推动情绪或证据，不做摆设。
3. **角色选择**：主角或关键配角做出一个无法完全撤回的动作。
4. **身体反应**：不是“我很难过”，而是手、胃、喉咙、脚步、呼吸、视线、沉默等可见反应。
5. **关系后果**：这一节结束后，谁和谁的关系、权力或信任发生变化。

## 文字质感规则

- 情绪可以直写，但后面必须接本场景独有动作、物件或身体反应。
- 对话必须带利益和关系位置。每句台词都要能回答“他/她为什么此刻这么说”。
- 不写通用漂亮话。不要用“她终于明白”“这意味着”“命运齿轮”“一切尽在不言中”替代场景。
- 不把读者该感受到的东西提前解释出来。让物件、动作、对话和选择自己完成情绪。
- 现实题材要有生活纹理：钱的来处、手续的麻烦、亲戚的称呼、单位/家庭场景的尴尬、公共场合的目光。
- 爽点必须有情绪债。没有前面的羞辱、误解、损失、犹豫或代价，后面的打脸只会像流程。

## 短篇写作失败信号

命中以下任一项，停止继续写下一节：

- 主角一直在等别人给证据，没有主动动作。
- 反派只是“坏”，没有具体欲望、利益和手段。
- 章节只是在解释前因，读者判断没有变化。
- 豪门、律师、老板、警察、系统或权威替主角完成价值证明。
- 情绪只靠标签和金句，没有生活物件和身体反应。
- 每节都有事件，但事件之间没有关系后果。
- 读完后只知道“发生了什么”，不知道“这对这个人意味着什么”。

## 通过标准

```text
Human resonance gate:
- Verdict: pass / revise / hard fail
- 共鸣经验：
- 关系压力：
- 生活物件：
- 主角主动选择：
- 情绪债与兑现：
- 文字质感风险：
- 必改动作：
```

`revise` 时先改 `设定.md` / `小节大纲.md`，再局部重写正文。`hard fail` 时保留用户确认的好开头、好句子和好物件，重建故事发动机。
