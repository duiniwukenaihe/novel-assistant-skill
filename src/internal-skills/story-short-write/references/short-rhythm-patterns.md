# 短篇节奏与爽点套路选择器

用途：素材卡、脑洞卡、人物关系和角色锁定卡确认后，进入小节大纲前，必须先选择短篇的主节奏模型、辅节奏模型、反转方式和兑现方式。它不是“套模板写烂梗”，而是先向读者承诺一种阅读快感，再让每一节都围绕这个承诺递进。

## 必须写入设定.md

```yaml
节奏模型:
  主节奏: R01_受辱蓄证公开打脸
  辅节奏: R06_亲情剥削自救边界
  读者承诺: 憋屈开场，证据翻盘，公开审判，主角完成自我定价
  反转方式: 证据反转 + 价值反转
  兑现方式: 公开场域打脸 + 关系切割 + 资产/尊严回收
  平台口径: 番茄短篇，快节奏，强情绪，低门槛，少解释
  防漂移边界: 外部人物只能提供压力/证据/镜面，不能替主角完成爽点
```

没有这块，不得进入 `小节大纲.md`。

## 读者承诺层

| 承诺 | 读者想要 | 常见风险 |
|---|---|---|
| 解气爽 / 打脸 | 看压迫者当众失控、付代价 | 外部权威代打，主角像旁观者 |
| 心死虐 / 追悔 | 看被辜负者彻底抽离，亏欠者迟到崩溃 | 只虐不反击，苦难堆砌 |
| 悬疑震撼 | 看真相一层层翻面 | 线索缺失，靠作者硬转 |
| 共鸣自救 | 看普通人从现实关系里挣脱 | 太说教，没有具体生活物件 |
| 甜宠治愈 | 看互相选择、互相托底 | 甜而无压，缺冲突 |
| 黑色幽默 / 反套路 | 看荒诞规则反噬提出规则的人 | 只搞笑，不推进人物选择 |

## 反转方式

- 身份反转：坐对面的人、真正的亲属、隐藏职业、真假身份翻面。
- 动机反转：看似恶意其实保护，或看似善意其实算计。
- 证据反转：录音、账本、转账记录、监控、聊天记录、旧物件重新定义事实。
- 关系反转：亲人不像亲人，外人反而提供镜面或证据。
- 视角反转：前文同一事件换视角后意义翻转。
- 时间线反转：现在的羞辱来自过去埋下的债。
- 价值反转：别人给主角定价，主角重新定义自己的价值。
- 规则反转：对方制定的规则被主角按原规则反杀。
- 舆论反转：群体从围攻主角转为审判压迫者。
- 物件回旋镖：小黑板、红包、账本、丝巾、旧钥匙、饭盒等反复出现并改变意义。

## 兑现方式

| 兑现 | 适用 | 注意 |
|---|---|---|
| 公开审判 | 打脸、群聊、相亲角、婚礼、直播 | 必须有证据，不靠嘴炮 |
| 金钱/资产回收 | 彩礼、房产、遗产、债务 | 账目要清楚，动机要可信 |
| 断亲/离婚/离场 | 原生家庭、婚恋压迫 | 要让读者相信“值得切割” |
| 火葬场追悔 | 虐恋、亲情悔痛 | 不原谅比复合更常见 |
| 法律/制度裁决 | 维权、职场、消费 | 程序提供压力，不替主角完成成长 |
| 舆论翻盘 | 热搜、群聊、直播 | 舆论只放大证据，不创造证据 |
| 自我定价/边界宣言 | 女性成长、世情 | 宣言前必须有行动铺垫 |
| 甜宠/治愈选择 | 甜宠、救赎 | 甜点要来自具体行为 |
| 代价收束 | 悬疑、复仇、心死 | 胜利要有余痛，避免太轻 |

## 平台口径

- 番茄/红果短篇：快节奏、强情绪、低门槛，前 300 字给压力，每节都要有可见变化。优先 R01/R02/R06/R07/R08/R20。
- 知乎盐选风：更适合自述、反转、悔痛、关系真相，可以给更多信息差和后劲。优先 R03/R04/R05/R10/R17。
- 七猫/点众风：节奏更直，冲突更显性，适合复仇、婚恋、家庭矛盾。优先 R01/R02/R07/R12。
- 短剧/短视频：公共场域、强动作、强对白、可视化道具更重要。优先 R01/R09/R20/R23。

## 使用门禁

1. 一篇短篇最多选一个主节奏、一个辅节奏。超过两个主模型会散。
2. 主节奏必须能拆出 3-5 个剧情爆点；拆不出，回到素材卡或设定重构。
3. 反转方式必须能在前文找到证据/物件/关系铺垫。
4. 兑现方式必须由主角的选择、证据、行动或价值宣言触发。
5. 外部权威、豪门、老板、警察、律师、热搜只能提供压力、程序、证据或镜面，不能替主角完成核心爽点。
6. `小节大纲.md` 每节必须标注：本节承接哪个节奏模型、推进哪个反转、兑现或铺垫哪个爽点。

## 在线方法论吸收备注

本选择器吸收了通用短篇平台方法论：短篇需要高能开篇、强情绪、快节奏和密集反转；短篇爽文常见链路是受辱、反击、打脸；番茄类平台偏好直白、快节奏、强情绪、低门槛。具体执行仍以本项目角色锁定卡、情节可行性门、人性共鸣门和短篇质量门为准。
